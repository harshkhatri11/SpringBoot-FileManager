## 🚀 Features

- ✅ Upload any file via REST API
- 📄 List all uploaded files
- 📥 Download individual files by filename
- 💾 Store files
- 🧱 Built with clean, modular Spring Boot architecture

## 🛠️ Tech Stack

- Java 21
- Spring Boot 3.5
- REST APIs
- Maven
